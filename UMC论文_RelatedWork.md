# UMC论文 - Related Work（相关工作）

## 2. Related Work

### 2.1 无监督多模态聚类

无监督多模态聚类旨在从无标签的多模态数据中发现潜在的语义结构，是机器学习领域的重要研究方向。早期的多模态聚类方法主要基于传统的聚类算法（如K-means、谱聚类）直接应用于拼接后的多模态特征，忽略了模态间的异构性和互补性。

近年来，深度学习方法在多模态聚类中取得了显著进展。**CC (Contrastive Clustering)** [引用] 通过对比学习学习多模态表示，使用InfoNCE损失最大化正样本对的相似度，最小化负样本对的相似度。然而，CC方法仅使用简单的特征拼接进行模态融合，未能充分利用模态间的交互信息。**MCN (Multimodal Clustering Network)** [引用] 引入了多模态编码器-解码器架构，通过重构损失学习多模态表示，但该方法对噪声敏感，在复杂对话场景中表现不稳定。

**SCCL (Self-supervised Contrastive Clustering Learning)** [引用] 结合了对比学习和聚类优化，通过自监督对比学习提升特征表示质量。然而，SCCL的模态融合策略仍然较为简单，且缺乏对训练过程的动态调整机制。**USNID (Unsupervised Semantic Network for Intent Discovery)** [引用] 专门针对意图发现任务设计，使用图神经网络建模样本间的关系，但该方法计算复杂度较高，且在多模态场景下的泛化能力有限。

与现有方法相比，UMC首次提出了ConFEDE双投影机制，通过分离主要信息和环境信息，显著提升了特征表示的丰富性。此外，UMC的文本引导注意力融合机制和自适应渐进式学习策略，为多模态聚类提供了更有效的训练范式。

### 2.2 多模态注意力融合机制

注意力机制在多模态学习中发挥着关键作用，能够自适应地关注不同模态的重要信息。**Cross-modal Attention** [引用] 通过计算不同模态间的注意力权重，实现模态间的信息交互。然而，传统的跨模态注意力通常是对称的，未能充分利用不同模态的信息密度差异。

**Text-guided Attention** [引用] 提出以文本为锚点引导其他模态的注意力计算，在视觉问答和图像描述生成任务中取得了良好效果。然而，这些方法主要应用于监督学习任务，在无监督聚类场景下的有效性尚未得到充分验证。**Gate-based Fusion** [引用] 使用门控机制自适应地加权不同模态的贡献，但现有方法通常使用固定的门控策略，缺乏对训练过程的动态调整。

UMC的文本引导多模态注意力融合机制创新性地将文本引导注意力应用于无监督聚类任务，通过双层注意力（交叉注意力+自注意力）和门控融合，实现了更有效的多模态特征融合。与现有方法相比，UMC的注意力机制专门针对聚类任务设计，能够更好地保持跨模态的语义一致性。

### 2.3 渐进式学习与自适应训练策略

渐进式学习（Curriculum Learning）是一种模仿人类学习过程的训练策略，通过从简单样本到复杂样本的渐进式训练，提升模型的学习效果。**Curriculum Learning** [引用] 在监督学习中取得了显著成功，但在无监督聚类中的应用相对较少。

**Self-paced Learning** [引用] 通过动态调整样本选择策略，优先选择高置信度的样本进行训练。然而，现有方法通常使用固定的阈值或简单的线性增长策略，未能充分考虑训练过程的动态性。**Adaptive Threshold Selection** [引用] 提出根据训练进度动态调整阈值，但该方法主要应用于单模态场景，在多模态聚类中的有效性尚未得到验证。

**Contrastive Learning with Pseudo-labels** [引用] 结合对比学习和伪标签学习，通过高质量的伪标签指导对比学习过程。然而，现有方法缺乏对伪标签质量的动态评估和调整机制，容易受到伪标签污染的影响。

UMC的自适应渐进式学习策略创新性地将S型曲线阈值增长与多维度自适应调整相结合，通过性能监控、损失分析和稳定性评估，实现了更智能的训练策略。与现有方法相比，UMC的渐进式学习策略专门针对多模态聚类任务设计，能够更好地处理不同数据集和不同训练阶段的动态变化。

### 2.4 对比学习在聚类中的应用

对比学习通过最大化正样本对的相似度、最小化负样本对的相似度，学习具有判别性的特征表示。**SimCLR** [引用] 和 **MoCo** [引用] 在视觉表示学习中取得了突破性进展，证明了对比学习的有效性。**SwAV** [引用] 将对比学习与聚类相结合，通过聚类中心作为负样本，提升了对比学习的效果。

在多模态场景下，**Multimodal Contrastive Learning** [引用] 通过跨模态对比学习学习多模态表示，但现有方法通常使用固定的对比学习策略，缺乏对样本质量的动态评估。**Clustering-guided Contrastive Learning** [引用] 使用聚类结果指导对比学习，但该方法容易陷入局部最优，且对初始聚类中心的选择敏感。

UMC将对比学习与自适应渐进式学习相结合，通过动态选择高质量样本进行监督学习，低质量样本进行无监督对比学习，实现了更有效的特征表示学习。与现有方法相比，UMC的对比学习策略能够更好地处理多模态场景下的样本质量差异，提升聚类效果。
